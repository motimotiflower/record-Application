# record-Application

## 概要
高校３年年生の課題研究の一環で、１年間かけて作成した初めてのアプリです。<br>
メモやタスクをカテゴリやファイルごとに保存することができます。<br>
AndroidStudioでJavaを用いて作成しました。

AIにいろいろ聞きながら、行き当たりばったりで作ったのでコードはお見苦しいものになっています。

## 使用技術
- Android Studio
- Java
- SQlite
- XML

2025年 2月　～　2026年 2月<br>
